# **Bus Reservation System - README**  

## **🚍 Project Overview**  
The **Bus Reservation System** is a database-driven solution for intercity bus services. It enables **online seat booking**, **real-time GPS tracking**, **maintenance management**, **driver scheduling**, and **passenger feedback collection**.  

---

## **✨ Key Features**  

### **🎟️ Online Seat Booking**  
- Secure seat reservations for intercity routes.  

### **📍 Real-time GPS Tracking**  
- Live bus location updates.  
- Estimated arrival time predictions.  

### **🔧 Maintenance Management**  
- Logs **mileage, engine checks, and incidents**.  
- Ensures vehicle safety and compliance.  

### **👨‍✈️ Driver Assignment**  
- Automated route and shift assignments.  

### **⭐ Passenger Feedback**  
- Ratings for **cleanliness, driver behavior, and punctuality**.  

---

## **🛠️ Technical Details**  
- **Database Type:** Relational (ERD Provided)  
- **Core Entities:**  
  - Buses, Routes, Drivers, Passengers  
  - Seats, Maintenance Logs, GPS Updates, Ratings  
- **Key Relationships:**  
  - Buses ↔ Maintenance Logs  
  - Passengers ↔ Seat Bookings  
  - Drivers ↔ Assigned Routes
